The development

The group assignment was loosely defined. Basically it was to create a web site, and use GitHub to coordinate the work flow.

First the project group got together and decided on what type of content we wanted on the site. For the main content we decided upon a small browser based game. We also wanted to create a page that describes the game mechanics, a page about JavaScripting, and an about page.

By using GitHub for coordination, we could easily divide the tasks between the group members, and thus work on different parts of the project at the same time. For example somebody could create the base HTML for the page, someone code the JavaScript needed for the game, and someone could create graphical elements.
